

HTML & CSS

A base de todo desenvolvimento Web é um bom HTML. Mais que aprender tagzinhas, é saber a 
importância de um código semântico e bem estruturado. Em cima disso, aplicamos o layout 
CSS que também é muito mais que apenas colocar uma cores e firulas por aí. É muito fácil 
escrever códigos HTML e CSS ruins. E é muito fácil não dar a atenção devida a esses 
fundamentos da Web. Mas não é o que recomendamos. Bons estudos!
